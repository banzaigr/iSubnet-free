> A fast, beautiful, and comprehensive network engineering toolkit — built for IT professionals, network engineers, and CCNA/CompTIA students.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📡 **IPv4 Calculator** | Compute network address, broadcast, wildcard, CIDR, host range, and binary representations in real-time |
| 🌐 **IPv6 Calculator** | Full 128-bit BigInt precision for prefix, range, and address type detection |
| ✂️ **Subnet Splitter** | Split any network into equal subnets or by host count using VLSM |
| 🔄 **Wildcard Converter** | Instantly convert between CIDR, subnet mask, and wildcard mask |
| 📚 **Reference Guide** | Built-in cheatsheet for IPv4 classes, RFC 1918 ranges, and IPv6 prefixes |
| 🗒️ **Notes** | Save and annotate calculation results directly inside the app |
| 🕓 **History** | Automatically tracks every calculation with one-click restore |

## 🚀 Getting Started

No installation required. The app runs entirely in the browser.

1. Clone or download this repository
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari)
3. That's it — no build step, no server, no dependencies

```bash
git clone https://github.com/risch72004/subnet-calculator.git
cd subnet-calculator
open index.html   # macOS
# or double-click index.html on Windows
```
